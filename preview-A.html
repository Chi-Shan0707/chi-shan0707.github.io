Featured Projects

token-verification-mirage

Solo-author · ICML 2026 AI4Math Workshop Poster

Controlled evaluation of token-level verification signals for LLM math reasoning. Shows that protocol choices like global pooling and direction-agnostic AUROC can shift reported performance by up to 0.18. Shallow token statistics are useful diagnostics, not stable standalone verifiers.

code-not-text

Solo measurement study · 31,040 runs across math, science, coding

Tests whether cheap hand-crafted features from reasoning traces predict correctness across domains. Strong math signal (AoA 0.958), partial science signal (0.799), weak coding transfer (0.434). The same feature family tracks convergence in math but not executable correctness in code.

TinyLoRA-GRPO-Coder

Solo project · RL for code generation

Small-parameter adaptation and GRPO training pipeline for verifiable competitive-programming code generation on Qwen2.5-Coder-3B. Uses compile-and-run rewards rather than static heuristics.
