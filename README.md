# About Yuhan Chi

Personal academic homepage for [Yuhan Chi](https://chi-shan0707.github.io/), built on AcademicPages / Minimal Mistakes and hosted with GitHub Pages.

## Site Content

- **Homepage**: selected research projects and community work.
- **CV**: current web CV at `/cv/`.
- **Portfolio**: project cards for selected open-source research systems.
- **Talks**: seminar notes and presentation materials.

## Notes

- Workshop and service links use official public pages when possible.
- Formal publications are listed only when there is a stable public venue or preprint record.
- The site keeps the original AcademicPages structure, but template examples and placeholder publications are removed from the public source tree.

## Local Preview

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The site will be available at `http://localhost:4000`.
